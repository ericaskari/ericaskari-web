## Getting Started

### Prerequisites

-   node
-   docker

### Installation

1. Clone the repo
    ```sh
    git clone https://github.com/ericaskari/ericaskari-web.git
    ```
2. Install NPM packages

    ```sh
    npm install
    ```

3. Start containers
    ```sh
    docker-compose up -d
    ```
4. Start backend

    ```sh
    npm run Start:App.Api
    ```

5. Start frontend
    ```sh
    npm run Start:App.Web
    ```
